User Profile Validation (Day 1 Challenge)
Description

This Python program checks whether a user profile is VALID or INVALID.
It takes four inputs from the user: Name, Email, Mobile Number, and Age.
Each input is checked using simple conditions.
___

Inputs :-

Full Name

Email ID

Mobile Number

Age
___

Validation Rules

Name :-

Must have at least 2 characters

Should not start or end with a space

Email :-

Must contain @ and .

@ should not be the first character

Mobile Number :-

Must be exactly 10 digits

Should contain only numbers

Should not start with 0

Age :-

Must be a number

Must be between 18 and 65
___

Working

The program checks each input one by one using if conditions.

String functions like len() and isdigit() are used.

If all conditions are satisfied, the output is VALID.

If any condition fails, the output is INVALID.
___

Output

User Profile is VALID

User Profile is INVALID
___

Language Used

Python
